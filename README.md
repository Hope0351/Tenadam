## Tenadam (ጤና አዳም)

## Installation Step

Clone a Project

- Run `composer install`
- Run `php artisan key:generate`
- Run `php artisan jwt:secret`
- **Setup .env file**
- **Run migration `php artisan migrate`**
- **Run seeder `php artisan db:seed`**
- **Create virtual host with name `local.tenadam.com`**
- **Run `npm install`**
- **Run `npm run dev`**
